# 🗨️ 실시간 채팅 웹앱 (Express + Socket.io)

이 프로젝트는 Node.js 기반의 실시간 채팅 웹앱입니다.  
사용자는 닉네임을 설정하고 채팅방에 입장할 수 있으며, 다른 사용자와 1:1 쪽지를 주고받을 수 있습니다.

---

1. 프로젝트 클론
```bash

cd 폴더명

node server.js

###  기술 스택
Node.js

Express

Socket.io

JSON 기반 채팅 로그 저장

##  주요 기능
✅ 닉네임 설정

✅ 방 입장/퇴장 알림

✅ 1:1 쪽지(DM) 기능
